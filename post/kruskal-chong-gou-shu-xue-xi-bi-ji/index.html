Kruskal 重构树学习笔记

2026-07-18

众所周知，最小生成树中有一个算法叫 Kruskal 算法。而 Kruskal 重构树正是 Kruskal 最小生成树算法的一个延伸。

我们回忆一下 Kruskal 最小生成树算法。每次我们会选取当前最短的可用边，只要这条边的两边没有连通那么就把这条边加入生成树。

那么 Kruskal 重构树构建的过程与上述过程极为类似。我们也是选取当前最短的可用边，这条边的两端没有连通的话，那么我们会给这条边新建成为一个点，然后再把这个点和边的两个端点所在的子树连接起来。

那么这样一颗 Kruskal 重构树有什么性质呢？

首先如果是按照上述的构建过程，我们明显能发现，这是一个大根堆。并且还满足原图中的点全部是叶子。非叶子节点代表的都是边。这两点是非常显然的。

他也有一些比较好的性质。

关键性质：原图中两个点之间所有路径中最大边权的最小值就是最小生成树上两点简单路径边权的最大值，也就是 Kruskal 重构树中两点 LCA 的点权

这一点我们稍微证明一下。我们考虑 Kruskal 最小生成树的构建过程，我们有没有可能走一些不在最小生成树中的边，反而让最大边权变小了呢？那么既然走了不在最小生成树中的边，让最大边权变小了，那么最大边权肯定就是不在最小生成树中的边提供的喽。而这样的边权却小于最小生成树中的边，那么肯定和最小生成树矛盾了，不然我可以选择这条边让最小生成树更小的。

这就是前半句话的解释。后半句话是显然的。

例题：P1967

纯模版题，就是求最大生成树下的 Kruskal 重构树的 LCA。

#include<bits/stdc++.h>
using namespace std;
int n, m;
const int M = 1e5 + 10, N = 2e4 + 10;
struct Edge {
    int u, v, val;
    bool operator < (const Edge &edge) const {
        return val > edge.val;
    }
}e[M];
int fa[N + M];
int findf(int x) {
    return (fa[x] == x) ? x : (fa[x] = findf(fa[x]));
}
int va[N + M];
vector<int>G[N + M];
void addedge(int u, int v) {
    G[u].push_back(v);
    G[v].push_back(u);
}
int st[N + M][20], dep[N + M];
void dfs(int u, int F) {
    st[u][0] = F; dep[u] = dep[F] + 1;
    // cout << u << ' ' << F << ' ' << dep[u] << ' ' << dep[F] << endl;
    for(int v : G[u]) {
        if(v == F) continue;
        dfs(v, u);
    }
}
int cnt;
void ST() {
    for(int i = 1; i < 20; ++i)
        for(int j = 1; j <= cnt; ++j)
            st[j][i] = st[st[j][i - 1]][i - 1];
}
int LCA(int x, int y) {
    if(findf(x) != findf(y)) return -1;
    if(dep[x] < dep[y]) swap(x, y);
    int u = dep[x] - dep[y];
    // puts("-");
    // cout << x << ' ' << y << endl;
    // cout << dep[8] << ' ' << dep[7] << endl;
    for(int i = 0; i < 20; ++i) 
        if((1 << i) & u) 
            x = st[x][i];
    // puts("=");
    // cout << x << ' ' << y << endl;
    if(x == y) return x;
    for(int i = 19; i >= 0; --i) 
        if(st[x][i] != st[y][i]) 
            x = st[x][i], y = st[y][i];
    // puts("!");
    // cout << x << ' ' << y << endl;
    return st[x][0];
}
int main() {
    ios::sync_with_stdio(), cin.tie();
    cin >> n >> m;
    for(int i = 1; i <= m; ++i) cin >> e[i].u >> e[i].v >> e[i].val;
    for(int i = 1; i <= n; ++i) fa[i] = i; cnt = n;
    sort(e + 1, e + m + 1);
    for(int i = 1; i <= m; ++i) {
        int fu = findf(e[i].u), fv = findf(e[i].v);
        if(fu != fv) {
            va[++cnt] = e[i].val;
            // cout << fu << ' ' << fv << ' ' << cnt << endl;
            addedge(fu, cnt);
            addedge(fv, cnt);
            fa[fu] = fa[fv] = fa[cnt] = cnt;
        }
    }
    for(int i = 1; i <= cnt; ++i) if(!dep[i]) dfs(findf(i), 0);
    ST();
    int Q;
    cin >> Q;
    while(Q--) {
        int x, y;
        cin >> x >> y;
        int lca = LCA(x, y);
        // cout << lca << endl;
        if(lca == -1) puts("-1");
        else cout << va[lca] << endl;
    }
    return 0;
}
