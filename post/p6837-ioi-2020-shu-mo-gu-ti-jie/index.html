P6837 [IOI 2020] 数蘑菇 题解

2026-07-14

好神的题啊。

首先 n≤20000n\leq20000n≤20000 这时候询问次数只能有 226226226 次，那么我们大概猜到这是一个 O(n)\mathcal{O}(\sqrt n)O(n​) 的东西（常数忽略）。

可是我连 O(n)\mathcal{O}(\sqrt{n})O(n​) 的方法都没想到（捂脸），但是我想到了个大概（明明只需要考虑一下 000 和 111 多的那个的。）

好的，那么我们首先讲一个朴素的 O(n)\mathcal{O}(\sqrt{n})O(n​) 的办法吧。

我们考虑一下如果我们已经知道 xxx 个 000，他们的位置分别是 a1,a2,a3,...,ana_1, a_2, a_3, ... ,a_na1​,a2​,a3​,...,an​ 那么我们可以做出这样的询问来得到 b1,b2,b3,...,bnb_1, b_2, b_3, ..., b_nb1​,b2​,b3​,...,bn​ 中 000 的个数。

我们可以问 b1,a1,b2,a2,b3,a3,...,bn,anb_1, a_1, b_2, a_2, b_3, a_3, ..., b_n, a_nb1​,a1​,b2​,a2​,b3​,a3​,...,bn​,an​ 这样的话由于一个夹着一个，那么我们可以得到 bbb 序列中 000 的个数。那么我们肯定是希望这样的 aaa 越多越好嘛。那我们怎么去求 aaa 呢？我们可以直接朴素。如果我们要求 kkk 个值的精确值。那么我们可以直接询问前 2k2k2k 个值， 0,ai0, a_{i}0,ai​ 这样去问，那么肯定 000 和 111 有一个数量大于等于 kkk。

于是我们的询问次数就来到 2k+nk2k+\frac{n}{k}2k+kn​ 当 k=n2k=\sqrt{\frac{n}{2}}k=2n​​ 的时候取到最小值。我们计算一下发现这个是明显大于 226226226 的。

因为数量级感觉是差不多的嘛。那么我们希望能对上面的过程进行一些优化。

我们可以想到，第二个步骤（一个夹一个）的步骤，除了第一个值，其他的如果跟左右不相等，贡献都是 222。 只有第一个贡献是 111。于是我们就可以直接得到第一个值的具体值。

既然如此，那么我们第一个步骤就可以一次问两个呀，如果我们知道了两个东西是 000。那么我们可以问 a1,0,a2,0a_1, 0, a_2, 0a1​,0,a2​,0 这样的话我们可以通过奇偶性判断 a1a_1a1​ 的值，然后再去判断 a2a_2a2​ 的值。

但是这样的优化还是只能让询问次数差不多是 k+nkk+\frac{n}{k}k+kn​ 接下来的优化就比较神了。

还是感觉这个第一个步骤实在太鸡肋。我们考虑如下的过程。

询问 1： a1,0,a2,0,a3,0a_1,0,a_2,0,a_3,0a1​,0,a2​,0,a3​,0

首先我们能根据奇偶性判断出来 a1a_1a1​ 的具体值。然后我们知道 a2a_2a2​ 和 a3a_3a3​ 中有几个 000。但是我们的目的是具体求出 a2a_2a2​ 和 a3a_3a3​ 呀。如果 a2a_2a2​ 和 a3a_3a3​ 不全为 000 或 111 的话，我们该怎么进一步去知道 a2a_2a2​ 和 a3a_3a3​ 谁是 000 谁是 111 呢？

询问 2：a4,0,a5,0,a2,0,1,a3,1a_4,0,a_5,0,a_2,0,1,a_3,1a4​,0,a5​,0,a2​,0,1,a3​,1

好，首先我们能根据奇偶性判断 a4a_4a4​ 的具体值。然后，如果 a2=0,a3=1a_2=0,a_3=1a2​=0,a3​=1 那么 a2,a3a_2,a_3a2​,a3​ 给答案的贡献就是 000，如果反过来呢， a2,a3a_2,a_3a2​,a3​ 给答案的贡献是 444 于是，我么可以通过除 444 的余数来判断 a5a_5a5​ 的具体值。然后我们可以通过判断答案是不是大于 444 判断 a2,a3a_2,a_3a2​,a3​ 的具体值。

那么我们可以想到，如果我们要求 kkk 个东西的具体值。那么我们大概只需要 45k+nk−o(1)\frac{4}{5}k+\frac{n}{k}-o(1)54​k+kn​−o(1) 次询问，最小值就大概是 245n−o(1)2\sqrt{\frac{4}{5}n}-o(1)254​n​−o(1) 差不多可以通过本题。

实现难度并不是很大。调参错了很多次，最后试出来 K = 111 的时候过了。

#include<bits/stdc++.h>
// #include "mushrooms.h"
using namespace std;
int use_machine(std::vector<int> x);
vector<int>b[2];
int count_mushrooms(int n) {
    int ans = 1;
    if(n <= 226) {
        for(int i = 1; i < n; ++i) ans += (use_machine({0, i}) ^ 1);
        return ans;
    }
    int K = 111;
    b[0].clear(), b[1].clear();
    b[0].push_back(0);
    #define cnt0 b[0].size()
    #define cnt1 b[1].size()
    #define id (cnt0 < cnt1)
    #define idsz max(b[0].size(), b[1].size())
    int len = 1;
    while(max(cnt0, cnt1) < 2) {
        int res = use_machine({0, len});
        b[res].push_back(len ++);
    }
    while(max(cnt0, cnt1) < 3) {
        int uid = id;
        int res = use_machine({len, b[uid][0], len + 1, b[uid][1]});
        if(res & 1) b[uid ^ 1].push_back(len);
        else b[uid].push_back(len);
        if(res >= 2) b[uid ^ 1].push_back(len + 1);
        else b[uid].push_back(len + 1);
        len += 2;
    }
    while(max(cnt0, cnt1) < K) {
        int uid = id;
        int res = use_machine({len, b[uid][0], len + 1, b[uid][1], len + 2, b[uid][2]});
        if(res & 1) b[uid ^ 1].push_back(len);
        else b[uid].push_back(len);
        res = res / 2;
        if(res == 0 || res == 2) {
            if(res == 0) {
                b[uid].push_back(len + 1);
                b[uid].push_back(len + 2);
            } else {
                b[uid ^ 1].push_back(len + 1);
                b[uid ^ 1].push_back(len + 2);
            }
            len += 3;
        } else {
            if(b[uid ^ 1].size() < 2) {
                int res2 = use_machine({len + 2, b[uid][0], len + 3, b[uid][1]});
                if(res2 & 1) b[uid].push_back(len + 1), b[uid ^ 1].push_back(len + 2);
                else b[uid].push_back(len + 2), b[uid ^ 1].push_back(len + 1);
                if(res2 >= 2) b[uid ^ 1].push_back(len + 3);
                else b[uid].push_back(len + 3);
                len += 4;
            } else {
                int res2 = use_machine({len + 3, b[uid][0], len + 4, b[uid][1], len + 1, b[uid][2], b[uid ^ 1][0], len + 2, b[uid ^ 1][1]});
                res2 --;
                if(res2 & 1) b[uid ^ 1].push_back(len + 3);
                else b[uid].push_back(len + 3);
                res2 /= 2;
                if(res2 & 1) b[uid ^ 1].push_back(len + 4);
                else b[uid].push_back(len + 4);
                if(res2 >= 2) b[uid].push_back(len + 2), b[uid ^ 1].push_back(len + 1);
                else b[uid].push_back(len + 1), b[uid ^ 1].push_back(len + 2);
                len += 5;
            }
        }
    }
    ans = b[0].size();
    vector<int>query;
    while(len < n) {
        query.clear();
        int i;
        int uid = id;
        for(i = 0; i < idsz && len + i < n; ++i) {
            query.push_back(i + len);
            query.push_back(b[uid][i]);
        }
        int A = use_machine(query);
        ans = ans + ((uid == 0) ? (query.size() / 2 - (A + 1) / 2) : ((A + 1) / 2));
        if(A & 1) b[uid ^ 1].push_back(len);
        else b[uid].push_back(len);
        len = len + i;
    }
    return ans;
    #undef cnt0
    #undef cnt1
    #undef idsz
    #undef id
}
